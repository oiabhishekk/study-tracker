GitHub Gist Cloud Sync

Sync progress across all your devices securely

Setup Instructions:

Step 1: Access Token
Go to GitHub Settings and click Generate new token (classic). Name it, check the gist box, click generate, and copy the token.
Step 2: Initialize Sync
Paste the token below, leave Gist ID empty, and click Create New Gist. The app will automatically initialize a private backup Gist.
Step 3: Connect Other Devices
On another device, open the app, paste the same Token and Gist ID, click Link Gist ID, and click 📥 Pull From Cloud.
Saved!
